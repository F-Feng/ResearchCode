o2scl::eos_had_temp_eden_base Class Referenceabstract

A hadronic EOS at finite temperature based on a function of the densities [abstract base]. More...

Detailed Description

This class provides automatic computation of calc_e() and calc_temp_e() assuming that calc_p() and calc_temp_p() are specified.
