Detailed Description

template<class fit_func_t = fit_funct11, class multi_func_t = uniform_prior<>, class vec_t = boost::numeric::ublas::vector<double>>
class o2scl::fit_bayes< fit_func_t, multi_func_t, vec_t >

This class is experimental.

This class uses Markov Chain Monte Carlo (MCMC) and marginal estimation to give a probability distribution for parameters in a fit.
