o2scl::quartic_real_gsl Class Reference

Solve a quartic with real coefficients and real roots (GSL) More...

Detailed Description

This class internally uses the GSL functions to solve the resolvent cubic and associated quadratics, while quartic_real_gsl2 contains explicit code to solve them instead.
