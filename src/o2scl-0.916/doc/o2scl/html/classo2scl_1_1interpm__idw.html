o2scl::interpm_idw< vec_t > Class Template Reference

Multi-dimensional interpolation by inverse distance weighting. More...

Detailed Description

template<class vec_t>
class o2scl::interpm_idw< vec_t >

This class performs interpolation on a multi-dimensional data set specified as a series of scattered points using the inverse distance-weighted average of the nearest three points. The function set_data() takes as input: the number of dimensions, the number of points which specify the data, and a "vector of vectors", e.g. std::vector<std::vector<double> > which contains the data for all the points.
