Detailed Description

template<class func_t>
class o2scl::inte_qawo_gsl_sin< func_t >

The integral

is computed for some frequency parameter , stored in inte_qawo_gsl_sin::omega .

An adaptive algorithm together with an series-acceleration method like that of inte_qags_gsl is used. Those subintervals with "large" widths where are computed using a 25-point Clenshaw-Curtis integration rule to handle the oscillatory behavior. In order to work efficiently, the Chebyshev moments for the particular weight function are computed in advance.

See GSL-based integration routines in the User's guide for general information about the GSL integration classes.
