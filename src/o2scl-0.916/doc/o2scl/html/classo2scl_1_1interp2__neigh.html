Detailed Description

template<class vec_t>
class o2scl::interp2_neigh< vec_t >

This class performs nearest-neighbor interpolation when the data points are not arranged in a specified order (i.e. not on a grid). For a set of data , the value of is predicted given a new value of x and y. Distances are determined with

The values and are specified in x_scale and y_scale, respectively. If these values are negative (the default) then they are computed with and .

This class stores pointers to the data, not a copy. The data can be changed between interpolations without an additional call to set_data(), but the scales may need to be recomputed with compute_scale().

The vector type can be any type with a suitably defined operator[].

Note
This class operates by performing a brute-force search to find the closest points.
