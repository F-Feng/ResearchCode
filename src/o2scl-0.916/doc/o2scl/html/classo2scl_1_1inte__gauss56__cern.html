Detailed Description

template<class func_t = funct11>
class o2scl::inte_gauss56_cern< func_t >

If is the 5-point approximation, and is the 6-point approximation to the integral, then integ_err() returns the result with uncertainty .

This class is based on the CERNLIB routines RGS56P and DGS56P which are documented at http://wwwasdoc.web.cern.ch/wwwasdoc/shortwrupsdir/d106/top.html
