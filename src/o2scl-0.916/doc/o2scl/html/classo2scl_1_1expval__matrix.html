o2scl::expval_matrix Class Reference

Matrix expectation value. More...

Detailed Description

See expval_base for some general notes on this and related classes.

This is a similar to expval_scalar, except that it allows updating and statistics for a set of matrices en masse. The data is stored internally in ublas matrix and tensor3 objects, but the public member functions operate with template types which are compatible with any vector class which provides double &operator[]. It is assumed that each call to add() contains a new measurement for all of the matrix entries.
